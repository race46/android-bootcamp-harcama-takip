# android-bootcamp-harcama-takip
## gdg icin hazrilanmis bitirme projesi
Uygulamamız harcamalarımızı takip edip kalan paramızı sürekli kontrol altında tutmak için tasarlandı. 
```bash
 Splash Screen
```
